Next Generation YouView

Next Generation YouView

Rethink, Redesign, Rebuild

Brief

Bring the YouView user experience into the modern day, utalizing a more cutting edge back end to incorperate the future of BT and TalkTalk TV packages for their customers

Client

BT TV and TalkTalk TV

Project Date

2018

Role

Product Designer

Overall Goal

The vision for the YouView project was a continuously updated product that was on the cutting edge of user experience and incorperating the latest back and front end technologies. With this in mind, the YouView team decided to not rest on the laurels of the successful launch but to start the process of the next generation.

There were a few high-end goals the company had which created our boundaries to work within

No Box Left Behind

Good, Better, Best

Powered By The Cloud

Every box since release had to support the next generation project

A philosophy that different hardware could have different levels of software to enable the higher end devices to have a more robust feature set

The whole project would be rewritten from Flash into the more recent HTML5 to allow easy in updating the software for fast iterative changes

Completely redesign of UI and UX

Youview is powered by its stakeholders and gave its users the wide range of content available to them. From the BBC to Channel 4 and Internet powered channels such as BT Sport, it was all brought to the forefront for the user. We made sure to bring this content front and center so the users could find the best content instantly

More visual content

For a visual redesign we wanted to ensure that the design was a engaging enticing element for the user. We wanted to create an easier to navigate interface with instantly recognisable features. We could also add responsive iconography relevent to the inputs of the user

Animations

Utalising HTML5 gave us the opportunity to investigate using CSS animations and Lottie Animation files to create visual interest and to give the user feedback to inputs from the remote control

Promotes content you want to see

The most important aspect for the user was the ability to see the content they love in the least amount of steps. The UI was redesign to be a much more visible design, no more reading a show name when there is a big content image making life much easier to find your favourites

Testing, Testing, Testing

User testing became a main focus company wide with the progression of this project. We were able to test hypothesis, interactive prototypes as well as the final product. We tested early and tested often

Analytics and data

The biggest change in the project came with the ability to gather and understand user data. In the previous version of YouView we had hardly any user data so decisions were not informed or made to the benefit of the user. By gathering anonymous data from the user we could evaluate and improve user flows to avoid pain points as well as learn what elements of a screen were not working

Iterative designs

The ability to utalise the front end of the hardware alloweed the design team to make fast itierative changes without the need for a developer to implement. We could take user feedback and make a HTML working prototype on the Hardware to showcase instantly to stakeholders
